# Happy Birthday Website 🎉

This is a simple birthday surprise website made using HTML, CSS, and JavaScript. It was created as a gift for Fari with messages and small pages from friends.

## What it does

- Shows a birthday greeting on the main page  
- Has links to different surprise pages  
- Includes messages from friends (Note, Ayesha, Charlotte)  
- Works on both mobile and desktop  
- Has a simple design with images and background  

## Files

- index.html → main page  
- style.css → styling  
- script.js → JavaScript  
- favicon.ico → website icon  
- fari.png → image used on homepage  
- note.html → message page  
- ayesha.html → message page  
- charlotte.html → message page  

## How to run

Just open `index.html` in any browser.
